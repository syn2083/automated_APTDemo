APT Demo Console

This is the APT Demo Console. From this application you will be able to set up and configure the Automated APT Demo system.

Demo Config

The Demo Config page contains settings related to the overall demo configuration such as;

  • APT Server Folder Locations
  • IDC Folder Locations
  • IDC Device Type(s)
  • IDC Target Scans/Hour
  • Trackdevice Folder Input Location

JIF Config

The JIF Config page contains settings related to the Job Control Files that will be automatically generated by the demo system. These settings cover job level manifest details along with target piece ranges (variable per job) and operator names.

Note: Target operator names must be configured in the APT Server before beginning the Demo!
